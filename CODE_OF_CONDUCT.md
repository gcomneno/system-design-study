# Code of Conduct

This repository follows a simple rule:

Be useful, respectful and concrete.

## Expected behavior

- Focus on improving the study material.
- Explain technical disagreements with reasons.
- Keep discussions practical and evidence-based.
- Respect other people's time and work.

## Unacceptable behavior

- Harassment or insults.
- Spam.
- Uploading unrelated files.
- Adding copyrighted material without permission.
- Adding video files, credentials or private data.

## Enforcement

The repository owner may remove issues, pull requests or content that violate these rules.
