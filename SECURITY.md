# Security Policy

This repository is a study repository and does not contain production software.

## Supported versions

Only the current `main` branch is maintained.

## Reporting security issues

If you notice accidental exposure of credentials, private data, tokens, copyrighted material, or sensitive files, open an issue or contact the repository owner privately.

## Sensitive material

Do not commit:

- API keys;
- tokens;
- passwords;
- personal data;
- private transcripts not meant for publication;
- downloaded videos or media files.

Run the repository hygiene checks before committing.
