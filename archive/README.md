# Archive

Directory riservata a note su materiale locale/privato usato come base di studio.

Non committare transcript integrali, corpus derivati da video, traduzioni complete o file scaricati senza licenza compatibile.

Per repository pubblico usare solo materiale rielaborato:

- note originali;
- quiz;
- esercizi;
- sintesi;
- risposte da colloquio;
- lezioni apprese.
