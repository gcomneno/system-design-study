# Repo Hygiene

## Regole

Questo repository contiene materiale didattico testuale per lo studio di System Design.

Sono esclusi deliberatamente:

- file video;
- download temporanei;
- file binari pesanti;
- cache locali;
- artefatti generati non necessari.

## Struttura

- `archive/`: materiale testuale grezzo o importato, usato come base di studio.
- `sessions/`: materiale didattico lavorato per singola sessione.
- `docs/`: convenzioni, note di manutenzione e regole del repository.
- `scripts/`: script di supporto.

## Nota copyright

Se il repository viene pubblicato, evitare di includere transcript completi o traduzioni integrali di materiale non proprio, salvo autorizzazione o licenza compatibile.

Per repository pubblico preferire:

- sintesi originali;
- quiz;
- domande da colloquio;
- esercizi;
- appunti rielaborati;
- riferimenti alle fonti senza riprodurre integralmente il contenuto.
