# Sessione 01 – API Design

## Obiettivo

Studiare i concetti fondamentali di API Design partendo dal transcript “What APIs are”.

## Concetti chiave

- API come contratto tra sistemi.
- API come astrazione dell’implementazione interna.
- API come confine tra servizi.
- Differenze pratiche tra REST, GraphQL e gRPC.
- Principi di design: consistency, simplicity, security, performance.
- Processo di progettazione: requisiti, scope, performance, sicurezza, sviluppo, deployment, manutenzione, deprecazione.
